# Move References

Move is an open source language for writing safe packages to manipulate on-chain objects (sometimes referred to as *smart contracts*). Move is a platform-agnostic language to enable common libraries, tooling, and developer communities across blockchains with vastly different data and execution models. Move is adaptable to meet the needs of the blockchain the code operates on, see [Move on Sui](/concepts/sui-move-concepts#differences) to review enhancements made to Move for optimization on the Sui blockchain.

* [Sui Framework](/references/framework): Provides `cargo doc`-generated content for core Sui modules.
* [The Move Book](https://move-book.com): A definitive guide for learning the Move language on Sui.
* [The Move Reference](https://move-book.com/reference): Architecture and syntax reference for the Move language.
