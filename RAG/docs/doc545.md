## สิ่งที่ต้องทำ

* เขียนเนื้อหา BCS สำหรับหัวข้อขั้นสูง
* สร้าง Docker image สำหรับหลากหลายแพลทฟอร์ม
* สร้างแบบทดสอบสำหรับแต่ละบท
* เพิ่มบทที่ห้าเกี่ยวกับมาตรฐานการแสดง NFT และการ implement SDK/frontend
